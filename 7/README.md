# Лабораторная работа №7: Многопоточность

## Задание 1: SumArray.java

Программа вычисляет сумму элементов массива в двух потоках.

- Каждый поток обрабатывает свою половину массива, затем результаты складываются в главном потоке.

## Задание 2: MatrixMax.java

Программа ищет наибольший элемент в матрице с использованием нескольких потоков.

- Каждый поток обрабатывает свою строку матрицы, затем результаты сравниваются.

## Задание 3: WarehouseTransfer.java

Программа с 3 грузчиками, которые переносят товары со склада.

- Грузчики могут переносить товары одновременно, но суммарный вес не может превышать 150 кг.
- Когда набрано 150 кг, грузчики отправляются на другой склад.
- Для синхронизации используется `CyclicBarrier`.
