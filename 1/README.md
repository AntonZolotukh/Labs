# Лабораторная работа №1: Базовые алгоритмы

## Задание 1: Primes.java

Программа находит и выводит все простые числа от 2 до 100.

- Метод `isPrime()` проверяет, есть ли у числа делители от 2 до квадратного корня.
- Если число имеет делитель, метод возвращает `false`, иначе — `true`.
- Запуск: `java Primes`

## Задание 2: Palindrome.java

Программа проверяет, являются ли строки палиндромами.

- `reverseString()` переворачивает строку, проходя по символам с конца в начало.
- `isPalindrome()` сравнивает исходную строку с перевёрнутой с помощью `equals()`.
- Запуск: `java Palindrome madam racecar apple kayak song noon`
