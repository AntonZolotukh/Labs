# Лабораторная работа №6: Коллекции

## Задание 1: TopWords.java

Программа считывает текстовый файл и выводит топ-10 самых часто встречающихся слов.

- Используется `Map<String, Integer>` для подсчёта частоты слов.
- Список сортируется по убыванию количества повторений.

## Задание 2: Stack.java

Обобщённый класс `Stack<T>`, реализующий стек на основе массива.

- `push()` — добавляет элемент в стек.
- `pop()` — удаляет и возвращает верхний элемент.
- `peek()` — возвращает верхний элемент без удаления.

## Задание 3: SalesTracker.java

Программа для учёта продаж в магазине.

- Используется `HashMap` для хранения пар «товар — количество продаж».
- Выводится общая сумма продаж и наиболее популярный товар.
