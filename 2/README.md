# Лабораторная работа №2: Объектно-ориентированное программирование (Вариант 6 – Геометрические фигуры)

## Задание: Shape.java, Sphere.java, Cylinder.java, RectangularPrism.java, Main.java

Программа демонстрирует принципы ООП на примере геометрических фигур.

- `Shape` — абстрактный базовый класс с полями `color`, `is3D` и статическим счётчиком `shapeCount`.
- `Sphere` — шар, добавляет поле `radius`. Переопределяет методы `getVolume()` и `getSurfaceArea()`.
- `Cylinder` — цилиндр, добавляет поля `radius` и `height`.
- `RectangularPrism` — прямоугольный параллелепипед, добавляет поля `length`, `width`, `height`.
- В классе `Main` создаются объекты разных фигур, выводится информация о каждой.
- Запуск: `java Main`
