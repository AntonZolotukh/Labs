# Лабораторная работа №5: Регулярные выражения

## Задание 1: NumberFinder.java

Программа ищет все числа в заданном тексте.

- Используется регулярное выражение `\\d+(?:\\.\\d+)?` для поиска целых и дробных чисел.

## Задание 2: PasswordValidator.java

Программа проверяет корректность пароля.

- Пароль должен состоять из латинских букв и цифр, быть длиной от 8 до 16 символов, содержать хотя бы одну заглавную букву и одну цифру.

## Задание 3: LowercaseUppercaseFinder.java

Программа находит все случаи, когда после строчной буквы идёт заглавная.

- Найденные фрагменты выделяются знаками «!» с двух сторон.

## Задание 4: IPValidator.java

Программа проверяет корректность ввода IP-адреса.

- IP-адрес должен состоять из 4 чисел, разделённых точками, каждое число от 0 до 255.

## Задание 5: WordsStartingWithLetter.java

Программа ищет все слова, начинающиеся с заданной буквы.

- Запуск: `java WordsStartingWithLetter`
