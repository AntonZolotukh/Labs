# Лабораторная работа №3: Хэш-таблицы (Вариант 6 – Телефонная книга)

## Задание 1: HashTable.java

Собственная реализация хэш-таблицы с разрешением коллизий методом цепочек.

- `put()` добавляет пару «ключ-значение» или обновляет значение, если ключ уже существует.
- `get()` возвращает значение по ключу.
- `remove()` удаляет пару по ключу.
- `size()` возвращает количество элементов.
- `isEmpty()` проверяет, пуста ли таблица.

## Задание 2: Contact.java, PhoneBook.java

Телефонная книга на основе хэш-таблицы.

- `Contact` — класс с полями `name`, `email`, `additionalInfo`.
- В `PhoneBook` создаются контакты, добавляются в телефонную книгу по номеру телефона, выполняется поиск и удаление.
- Запуск: `java PhoneBook`
